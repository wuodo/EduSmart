# EduSmart Local Network Access Guide

## For Satellite Users

### How to Access the System

1. **Connect to the same WiFi network** as the EduSmart computer
2. **Open your web browser** (Chrome, Firefox, Safari, etc.)
3. **Enter this URL**: `http://192.168.10.43:3000`
4. **Start using the system!**

### What You Can Access

- **Marketing Dashboard**: View and manage inquiries
- **Follow-ups**: Schedule and track follow-up activities
- **Student Management**: Handle student applications and data
- **Reports**: Generate various reports and analytics

### Troubleshooting

**If you can't access the system:**

1. **Check your WiFi connection** - Make sure you're connected to the same network
2. **Try the IP address**: `192.168.10.43:3000`
3. **Ask the administrator** if the system is running
4. **Check if your device is on the same network** (should start with 192.168.10.x)

### System Requirements

- Any modern web browser
- Internet connection (for initial loading)
- Access to the local network

### Support

If you have any issues, contact the system administrator.

---

**Note**: This system is only accessible when connected to the local network and when the EduSmart computer is running.
