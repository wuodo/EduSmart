# EduSmart CRM — Sales Team Training Guide

## Welcome to EduSmart CRM

EduSmart CRM is your **all-in-one sales platform** for managing prospective students from first inquiry through to admission. Every lead, follow-up, and admission letter lives in one place — no more spreadsheets, sticky notes, or missed follow-ups.

---

## 1. Your Dashboard (Home Page)

When you log in, the **Dashboard** gives you a snapshot of your day:

| Card | What it tells you |
|-----|-------------------|
| **Hot Leads** | High-priority inquiries needing immediate action |
| **Overdue** | Follow-ups past their scheduled date |
| **New Today** | Fresh inquiries that came in today |
| **Conversion %** | Your personal conversion rate |
| **Focus Score** | Overall performance indicator |

**Daily Briefing Banner** — a teal strip at the top shows your greeting, stats, and a priority message like *"You have 3 overdue follow-ups — start with those."*

## 2. Inquiries (Your Lead List)

**Where:** Sidebar → **Inquiries**

This is your main workspace. Every prospective student appears here as a row.

### Columns You See

| Column | What it means |
|--------|---------------|
| **#** | Row number |
| **Name** | Prospect's full name |
| **Phone / Email** | Contact info |
| **Program** | Course they applied for |
| **Source** | Where they came from (Website, Referral, Walk-in, etc.) |
| **Status** | Current stage (Hot, Warm, Cold, etc.) |
| **Score** | Lead quality score (0-100) |
| **Assigned To** | Who's handling this lead |

**Green highlighted rows** = New inquiries from the website that haven't been viewed yet. Click to open and the green fades.

### Quick Actions

| Icon | Action |
|------|--------|
| 👁️ | View full inquiry details |
| ✏️ | Edit inquiry fields |
| 📞 | Call the prospect |
| ✉️ | Send email |
| 💬 | Open WhatsApp chat |
| 🗑️ | Delete (with permission) |

### Filtering & Searching

Use the **filter bar** above the table to narrow down leads:

- **Search** — type any name, phone, or email
- **Status** — filter by Hot / Warm / Cold
- **Source** — Website, Facebook, Referral, etc.
- **Owner** — filter by assigned staff member (admins only)
- **More filters** — county, program, KCSE grade, intake, gender, payment status

## 3. Lead Status Lifecycle

Every lead moves through these stages:

```
New → Contacted → Qualified → Proposal → Negotiated → Won / Lost
```

| Status | What it means | Your action |
|--------|---------------|-------------|
| **New** | Fresh lead, not yet contacted | First contact within 24h |
| **Contacted** | You've reached out | Qualify the lead |
| **Qualified** | Interested, meets criteria | Send proposal/admission letter |
| **Proposal** | Letter sent | Follow up |
| **Negotiated** | Discussing terms | Close |
| **Won** | Student registered | Celebrate! |
| **Lost** | Not interested | Archive |

**Stage-Gate Rules:** Some transitions require fields to be filled. For example, moving from `Contacted → Qualified` requires KCSE grade and program of interest. This ensures data quality.

## 4. Follow-ups (Your Task List)

**Where:** Sidebar → **Follow-ups**

Every scheduled call, SMS, WhatsApp, or email with a prospect is a follow-up.

### Creating a Follow-up

1. Click **Schedule Follow-up** button
2. Select the **inquiry** (prospect)
3. Choose **type** (Call, SMS, WhatsApp, Email)
4. Set **date & time**
5. Add **notes** about what to discuss
6. Click **Save**

### Follow-up Statuses

| Status | Meaning |
|--------|---------|
| **Pending** | Scheduled, not yet done |
| **Completed** | Done — outcome recorded |
| **Rescheduled** | Moved to a later date |
| **Cancelled** | No longer needed |

### Overdue Follow-ups

When a follow-up passes its scheduled date, it becomes **overdue**. The system:
- Shows it in red on your dashboard
- Sends you a **notification bell** alert
- Escalates to your manager after **48 hours**

**Pro tip:** Start your day by clearing overdue follow-ups first — they're the hottest leads going cold.

## 6. Calendar (Your Schedule)

**Where:** Sidebar → **Calendar**

All your follow-ups and tasks displayed in month/week/day views.

### Features
- **Month view** — overview of the month
- **Week view** — your week at a glance
- **Day view** — hourly time slots (7am-8pm)
- **Drag to reschedule** — drag an event to a different day or time
- **Click any slot** — create a new follow-up directly from the calendar
- **Color-coded** — Calls = teal, SMS = indigo, WhatsApp = green, Email = blue

### Stats Cards
- **Today** — events scheduled for today
- **Overdue** — past-due events
- **Pending** — upcoming events
- **Completed** — done

### Features

## 8. Admission Letters

**Where:** Sidebar → **ADM Letters**

Generate and send admission letters to qualified prospects.

### Steps
1. Enter the **Admission Date** (auto-filled to 30 days from now)
2. Select a **Template** (default, short, or detailed)
3. Click **Generate** — letter is created with reference number
4. **Send via Email** — opens compose modal with PDF auto-attached
5. **Share via WhatsApp** — send the letter directly
6. **Download / Preview** — review before sending

### Letter Statuses

| Status | Meaning |
|--------|---------|
| **Not Generated** | No letter created yet |
| **Generated** | Letter created |
| **Sending** | Email being sent |
| **Sent** | Emailed to prospect |
| **Downloaded** | Prospect downloaded PDF |
| **Acknowledged** | Prospect confirmed receipt |
| **Signed** | E-signed online (if enabled) |

## 9. Notifications & Alerts

**Bell icon** in the top header bar (red badge):

| Notification | When it appears |
|-------------|----------------|
| **New Inquiry Assigned** | A lead is assigned to you via round-robin |
| **New Website Inquiry** | Someone submitted a form on the website (unassigned) |
| **SLA Breach Alert** | A lead has no response for 48+ hours |
| **Escalation** | A follow-up is 48h overdue — manager is copied |
| **Follow-up Reminder** | A scheduled follow-up is due or overdue |

Click the bell to view details, then **"View details →"** to jump to the inquiry.

## 11. Workflows (Automation)

**Where:** Sidebar → **Workflows**

The system can automatically handle repetitive tasks. Your admin sets these up, but here's what they can do:

| Trigger | Example Auto-Action |
|---------|-------------------|
| New inquiry with status "Hot" | Auto-schedule a call follow-up in 2 days |
| Status changes from "Warm" → "Hot" | Auto-add tag "priority-follow-up" |
| Follow-up completed | Auto-schedule next step (e.g., WhatsApp in 3 days) |

You don't need to manually create these — the system does it for you.

## 12. QA Review (Data Quality)

**Where:** Sidebar → **QA Review**

The system automatically checks for incomplete data and flags it for review. Items appear here when:
- A lead is missing email, phone, or program
- A lead has no follow-up scheduled
- A reassignment happens

**Your role:** Review flagged items → approve or reject with a comment.

## 15. Daily Workflow (Recommended)

### Morning (First 15 minutes)

1. ✅ **Check Dashboard** — see your briefing banner (hot leads, overdue, new today)
2. ✅ **Clear overdue follow-ups** — check the Follow-ups page
3. ✅ **Process new inquiries** — click on green-highlighted rows
4. ✅ **Check notifications** — click the red bell badge

### Throughout the Day

5. ✅ **Log every interaction** — create follow-ups after every call/email/SMS
6. ✅ **Update lead status** — move leads through the pipeline as they progress
7. ✅ **Generate admission letters** — for qualified prospects
8. ✅ **Use email templates** — for faster responses

### End of Day

9. ✅ **Review tomorrow's schedule** — check the Calendar
10. ✅ **Complete pending follow-ups** — don't let them become overdue
